## 📌 Objetivo del proyecto
Desarrollar un programa en Java que permita trabajar con arreglos unidimensionales, realizando operaciones básicas como ingreso de datos, cálculo de diferencias y obtención del promedio.

## 🧠 Descripción
El programa realiza las siguientes acciones:
- Define dos arreglos unidimensionales de 7 posiciones.
- Solicita al usuario ingresar valores numéricos para cada arreglo.
- Llena un tercer arreglo con la diferencia entre los dos primeros.
- Calcula el promedio de todos los datos almacenados.
- Muestra los valores del tercer arreglo y el promedio final.
